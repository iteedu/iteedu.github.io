    6.3、把下列C语句的语句改写成功能相同的汇编语言程序片段(其中：变量都为整型变量)。

    1)、h = (key & 0XFF00) >> 8;

    2)、k = (k + '1' – 0xabcd) / 56;

    3)、for (i = s = 0; i < 100; i++) s += data[i] * 2;

    4)、for (s = 0, i = 100; i > 0; i--) s += i * 2;

    6.4、把下列C语言的语句改写成等价的汇编语言程序段(不考虑运算过程中的溢出)。

    If (a<1 || b/4 > 10 && c%8==5) {

    a = 20+b++;

    c <<= 2;

    }

    else {a = 21-(++c);

    b--;

    }

    其中：变量a、b和c都是有符号的整型(int)变量。
