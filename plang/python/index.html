    Python

    因为简单所以喜爱。

    我用python的感觉就想用汉语写文章，从来没有遇见过什么语法上的问题。想到哪写到哪，想出来就写完了。呵呵。

    其实再怎么说一门语言好也是没用的，必须自己用一下才行。

    至于现在发展的势头如何，我只说几点就行了。

    google有三大公认的语言，python是其中的一种。python被google作为网络web开发的语言，从google的app是用python作为开发语言就可以看到。

    python已集成到linux中，作为一种必备的工具加入。

    vista的原型开发中很多用python开发。

    豆瓣网是用python开发的。

    另外python在游戏开发，仿真等很多方面都有广泛应用。这里就不多说了。

    python是初学者的最好的入门语言，在欧洲很多学校都用python来教编程。

    python是开发人员最得心应手的工具。
