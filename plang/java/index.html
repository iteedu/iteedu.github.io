    • JAVA

    不得不承认，JAVA已成为工业化“生产”程序的标准语言。从应用软件，游戏开发到WEB与嵌入式，到处都是JAVA的身影。如果你还对JAVA疑问，请看随笔“JAVA为什么成为工业化标准“。

    总起来说JAVA在应用软件开发、WEB开发和游戏开发方面已是很有建树。而且已成为一种标准（看JAVA为什么成为工业化标准）。是一种纯面向对象设计的语言，也因为它的纯，使它学起来很简单。

    J2SE、J2ME、J2EE三大平台都有强大的库支持和大公司撑腰。虚拟机使跨平台变的很自然。

    现在eclipse和netbeans这些IDE的推出使JAVA编程日益变的简单。

    ROSE等UML建模工具对JAVA的支持又使JAVA锦上添花，现在已经可以到除方法代码外的一切代码都由工具自动生成的地步了。

    在WEB开发方面，J2EE体系众多的框架使编程变得异常简单。手机软件和游戏的开发也日渐出现更多JAVA的身影。

    貌似JAVA现在是如日中天，便不要忘记语言只是工具。你可能用一个月学会JAVA，但要用一辈一学编程。

    不要迷信于语言本身。现在JAVA是最容易让我们迷惑的了。庞大的库让我们费尽心血，到头来还是一个程序都写不出来。

    怎么学编程是很重要的。不要忘记学的是编程，而不是语言。
