    6.49. Man-DB-2.5.6

    Man-DB 包含查找和显示 man 手册页的程序。

    预计编译时间: 0.4 SBU
    所需磁盘空间: 22 MB

    6.49.1. 安装 Man-DB

    为编译 Man-DB 做准备：

    ./configure --prefix=/usr --libexecdir=/usr/lib \
    --sysconfdir=/etc --disable-setuid \
    --with-browser=/usr/bin/lynx --with-vgrind=/usr/bin/vgrind \
    --with-grap=/usr/bin/grap

    配置选项的含义：

    --disable-setuid

    man 不给用户 man 设置uid位。

    --with-...

    这三个选项用来设置一些默认的程序。 lynx 是一个基于文本的web浏览器 (安装说明请参考 BLFS）， vgrind 将程序源代码转变为Groff的输入， grap 对Groff文档的排版图很有用。 vgrind和 grap 程序对于查看手册页并不是必须的。它们不是 LFS 或 BLFS的一部分， 如果你愿意，你可以在完成LFS系统后，自己安装这些程序。

    编译这个软件包：

    make

    要测试结果，请运行：

    make check

    安装这个软件包：

    make install

    6.49.3. Man-DB 的内容

    安装的程序: accessdb, apropos (link to whatis), catman, lexgrog, man, mandb, manpath, whatis, and zsoelim

    简要描述

    accessdb

    将 whatis 数据库的内容转储为人类可读形式

    apropos

    搜索 whatis数据库，显示包含给定字符串的系统命令的简短描述。

    catman

    创建或更新预格式化的 manual 页

    lexgrog

    显示一行给定 manual 页的摘要信息。

    man

    格式化并显示请求的 manual 页

    mandb

    创建或升级whatis数据库

    manpath

    显示 $MANPATH 的内容或在man.conf中设置的搜索路径（如果 $MANPATH 没有设置），以及用户的环境变量。

    whatis

    搜索whatis数据库，显示包含给定关键字的系统命令的简短描述。

    zsoelim

    读取文件并用提到的 file的内容来替换 .so file 格式的行。
