    • Enumerator 对象

    启用枚举集合中的项目。

    enumObj = new Enumerator([collection])

    参数

    enumObj

    必选项。要赋值为 Enumerator 对象的变量名。

    collection

    可选项。任意 Collection 对象。

    说明

    集合与数组的不同点在于集合的成员不能直接访问。不象处理数组时使用下标，这时只能将当前项目指针移动到集合的下一或前一元素。

    Enumerator 对象提供了访问集合的任何成员的方法，其操作与 VBScript 中的 For...Each 语句相似。

    示例

    下面的代码显示了 Enumerator 对象的用法: 

    function ShowDriveList(){
   var fso, s, n, e, x;                     // 声明变量。
   fso = new ActiveXObject("Scripting.FileSystemObject");
   e = new Enumerator(fso.Drives);          // 在驱动器上创建 Enumerator。
   s = "";
   for (;!e.atEnd();e.moveNext())           // 枚举驱动器集合。
      {
         x = e.item();
         s = s + x.DriveLetter;
         s += " - ";
         if (x.DriveType == 3)              // 查看是否网络驱动。
            n = x.ShareName;                // 获取共享名。
         else if (x.IsReady)                // 查看驱动器是否已就绪。
            n = x.VolumeName;               // 获取卷名。
         else
            n = "[驱动器未就绪]";
         s +=  n + "<br>";
      }
   return(s);                               // 返回活动驱动器列表。
}

    属性

    Enumerator 对象没有属性。

    方法

    atEnd 方法 | item 方法 | moveFirst 方法 | moveNext 方法

    要求

    版本 3

    请参阅

    Drives 集合 | Files 集合 | Folders 集合
