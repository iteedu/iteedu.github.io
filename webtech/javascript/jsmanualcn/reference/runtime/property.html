    • 属性

    语言元素
    		 描述
    AtEndOfLine 属性 如果在 TextStream文件中文件指针紧接在 end-of-line 标志之前则返回 True ；否则返回 False。
    AtEndOfStream 属性 如果在 TextStream 文件中文件指针到达文件尾则返回 True ；否则返回 False 。
    Attributes 属性 设置或返回文件或文件夹的属性。
    AvailableSpace 属性 向用户返回指定驱动器或网络共享上的可用空间大小。
    Column 属性 返回 TextStream 文件中当前字符所在的列号。
    CompareMode 属性 设置和返回在 Dictionary 对象中对字符串键进行比较的比较方式。
    Count 属性 返回集合或 Dictionary 对象中的项目数。
    DateCreated 属性 返回指定文件或文件夹创建的日期和时间。
    DateLastAccessed 属性 返回指定文件或文件夹最后一次访问的日期和时间。
    DateLastModified 属性 返回指定文件或文件夹最后一次修改的日期和时间。
    Drive 属性 返回指定文件或文件夹所在驱动器的驱动器号。
    DriveLetter 属性 返回本地物理驱动器或网络共享的驱动器号。
    Drives 属性 返回一个 Drives 集合，由所有本地机器上可用的 Drive 对象组成 。
    DriveType 属性 返回说明指定驱动器类型的值。
    Files 属性 返回一个 Files 集合，由所有包含在指定文件夹中的 File 对象组成，包括那些设置了隐藏和系统文件属性的文件。
    FileSystem 属性 返回指定驱动器所使用的文件系统类型。
    FreeSpace 属性 向用户返回指定驱动器或网络共享上可用空间的大小。
    IsReady 属性 如果指定驱动器已就绪则返回 True ；否则返回 False 。
    IsRootFolder 属性 如果指定文件夹是根文件夹则返回 True ；否则返回 False 。
    Item 属性 根据 Dictionary 对象中指定的 key 设置或返回 item 。
    Key 属性 在 Dictionary 对象中设置 key 。
    Line 属性 返回 TextStream 文件中的当前行号。
    Name 属性 设置或返回指定文件或文件夹的名称。
    ParentFolder 属性 返回指定文件或文件夹的父文件夹对象。
    Path 属性 返回指定文件、文件夹或驱动器的路径。
    RootFolder 属性 返回表示指定驱动器上根文件夹的 Folder 对象。
    SerialNumber 属性 返回用于唯一标识驱动器卷的数字序列号。
    ShareName 属性 返回指定驱动器的网络共享名。
    ShortName 属性 返回一些程序使用的短名称，这些短名由需要以前的 8.3 命名规范的程序使用。
    ShortPath 属性 返回一些程序使用的短路径，这些短路径由需要以前的 8.3 文件名规范的程序使用。
    Size 属性 以字节为单位返回指定文件或文件夹的大小。
    SubFolders 属性 返回由指定文件夹中包含的所有文件夹组成的 Folders 集合，包括那些设置了隐藏和系统文件属性的文件夹。
    TotalSize 属性 以字节为单位返回驱动器或网络共享的总共空间大小。
    Type 属性 返回关于文件或文件夹类型的信息。
    VolumeName 属性 设置或返回指定驱动器的卷号。
