    • 方法

    语言元素
    		 描述
    Add 方法 (Dictionary) 向 Dictionary 对象中添加一对键和项目。
    Add 方法 (Folders) 向 Folders 集合中添加新的 Folder 。
    BuildPath 方法 在已有路径后追加一个名称。
    Close 方法 关闭打开的 TextStream 文件。
    Copy 方法 将指定文件或文件夹从一个位置复制到另一个位置。
    CopyFile 方法 从一个位置向另一个位置复制一个或多个文件。
    CopyFolder 方法 从一个位置向另一个位置复制整个文件夹。
    CreateFolder 方法 创建文件夹。
    CreateTextFile 方法 创建指定的文件名并返回可用于读写该文件的 TextStream 对象。
    Delete 方法 删除指定文件或文件夹。
    DeleteFile 方法 删除指定文件。
    DeleteFolder 方法 删除指定文件夹及其内容。
    DriveExists 方法 如果指定驱动器存在则返回 True ；如果不存在则返回 False 。
    Exists 方法 如果 Dictionary 对象中存在指定键则返回 True ，否则返回 False 。
    FileExists 方法 如果指定文件存在则返回 True ；否则返回 False 。
    FolderExists 方法 如果指定文件夹存在则返回 True ;否则返回 False 。
    GetAbsolutePathName 方法 根据所提供的路径说明返回完整明确的路径。
    GetBaseName 方法 根据提供的路径说明返回一个字符串，包含文件或文件夹的基本名称（不包括任何文件扩展名）。
    GetDrive 方法 返回相应于指定路径中驱动器的 Drive 对象。
    GetDriveName 方法 根据指定路径返回包含驱动器名称的字符串。
    GetExtensionName 方法 返回一个字符串，包含路径中最后成分的名称。
    GetFile 方法 返回相应于指定路径中文件的 File 对象。
    GetFileName 方法 返回指定路径中的最后文件名或文件夹名，该路径不能是驱动器说明。
    GetFileVersion 方法 返回指定文件的版本号码。
    GetFolder 方法 返回相应于指定路径中文件夹的 Folder 对象。
    GetParentFolderName 方法 返回一个字符串，包含指定路径中最后文件或文件夹的父文件夹名。
    GetSpecialFolder 方法 返回指定的特殊文件夹。
    GetTempName 方法 返回一个随机生成的临时文件或文件夹名称。
    Items 方法 返回包含 Dictionary 对象中所有项目的数组。
    Keys 方法 返回包含了 Dictionary 对象中全部已有键的数组。
    Move 方法 将指定文件或文件夹从一个位置移动到另一个位置。
    MoveFile 方法 从一个位置向另一个位置移动一个或多个文件。
    MoveFolder 方法 从一个位置向另一个位置移动一个或多个文件夹。
    OpenAsTextStream 方法 打开指定文件并返回可用于读、写或追加该文件的 TextStream 对象。
    OpenTextFile 方法 打开指定文件并返回可用于读、写或追加该文件的 TextStream 对象。
    Read 方法 从 TextStream 文件中读取指定数量的字符并返回结果字符串。
    ReadAll 方法 读取整个 TextStream 文件并返回结果字符串。
    ReadLine 方法 从 TextStream 文件中读取整行（在 newline 字符前，但不包括它）并返回结果字符串。
    Remove 方法 从 Dictionary 对象中删除一个键和项目对。
    RemoveAll 方法 删除 Dictionary 对象中的所有键和项目对。
    Skip 方法 在读取 TextStream 文件时掠过指定数目的字符。
    SkipLine 方法 在读取 TextStream 文件时掠过下一行。
    Write 方法 向 TextStream 文件中写入指定字符串。
    WriteBlankLines 方法 向 TextStream 文件中写入指定数量的 newline 字符。
    WriteLine 方法 向 TextStream 文件中写入指定字符串和 newline 字符。
