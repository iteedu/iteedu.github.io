    时间单位 Time Units
    s     ms
    长度单位 Length Units
    em    ex    px    pt    pc    in    cm    mm
    频率单位 Frequency Units
    kHz    Hz
    颜色单位 Color Units
    #RRGGBB    rgb ( R,G,B )    Color Name

    deg    grad    rad
