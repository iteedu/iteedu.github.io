    Positioning Properties
    属性    		CSS Version
    版本    		Compatibility
    兼容性    		Inherit From Parent
    继承性    		 Description
    简介
    positionCSS2IE4+ , NS4+无检索对象的定位方式
    z-indexCSS2IE4+ , NS6+无检索或设置对象的层叠顺序
    topCSS2IE4+ , NS4+无检索或设置对象与其最近一个定位的父对象顶部相关的位置
    rightCSS2IE5+无检索或设置对象与其最近一个定位的父对象右边相关的位置
    bottomCSS2IE5+无检索或设置对象与其最近一个定位的父对象底边相关的位置
    leftCSS2IE4+ , NS4+无检索或设置对象与其最近一个定位的父对象左边相关的位置

     
     
     
    说明：这种颜色是CSS2标准属性。这种颜色是IE建议样式表属性。这种颜色是NS私有属性。这种颜色是目前尚无浏览器支持的属性。
