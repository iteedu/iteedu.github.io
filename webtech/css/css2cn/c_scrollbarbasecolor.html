    scrollbar-base-color版本：IE5.5+专有属性　继承性：有

     
    语法：
     
    scrollbar-base-color : color
     
    参数：
     
    color : 　指定颜色。请参阅颜色单位和附录：颜色表
     
    说明：
     
    设置或检索滚动条基准颜色。其它界面颜色将据此自动调整。
    请参阅overflow属性 。
    对应的脚本特性为scrollbarBaseColor。请参阅我编写的其他书目。
     
    示例：
     
    div {scrollbar-base-color : buttonface; }
     
    请您用下面的按钮选择这段文字的scrollbar-base-color属性的值。看一看会发生什么，然后您就会明白这个属性的意义。希望您喜欢这本电子书。谢谢。
