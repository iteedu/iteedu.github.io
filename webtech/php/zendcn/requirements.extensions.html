    • A.2. PHP 扩展

    下面列出了所有 PHP 内建支持的扩展，以及它们在 Zend Framework 中如何应用。 这个可以帮助了解哪个扩展需要在应用中使用。 不是每一个被 Zend Framework 使用的扩展都要应用在程序中。

    使用方式“依赖”表示当扩展不可用时，组件或者类也不可使用。 使用方式“可选”表示当扩展不可用时，组件或者类自动使用代替方案。
