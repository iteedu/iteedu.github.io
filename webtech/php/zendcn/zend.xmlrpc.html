    62.1. 简介

    从 XML-RPC 主页上的描述可以了解到，XML-PRC 是 “使用 HTTP 作为传输协议、XML 作为编码方式的远程过程调用。在 XML-RPC 被设计得尽量简单的同时，又允许传递、处理和返回复杂的数据结构。”

    Zend Framework 提供了调用远程 XML-RPC 服务或创建新的 XML-RPC 服务两种功能。
