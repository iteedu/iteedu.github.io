    CXV. Printer Functions

    简介

    These functions are only available under Windows 9.x, ME, NT4 and 2000. They have been added in PHP 4.0.4.

    安装

    This PECL extension is not bundled with PHP.

    Windows users must enable php_printer.dll inside of php.ini in order to use these functions. 可以从不同的 PECL 页面（选择适合自己 PHP 版本的库）取得未绑定的 PECL 扩展：PECL for PHP 4.3.x，PECL for PHP 5.0.x 或 PECL Unstable。

    运行时配置

    这些函数的行为受 php.ini 的影响。
