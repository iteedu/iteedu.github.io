    PHP

    站长原创，版权所有ITEEDU

    有人说在J2EE、.NET、Python、ruby等出现后，PHP将走向衰落。其实不然，只要一点就可以说明PHP将更好的走下去，那就是大量的用户和编程人员。这是一个用户至上的时代，只要有大量的用户存在没有活不下去的理由。随着PHP版本的升级换代，PHP也有了面向对象等高级特性。最多的用户，最成熟稳定的技术，最好找的空间类型，众多的应用，我相信PHP会一直走下去。

    PHP的主要特点：

    跟Apache服务器紧密结合

    不断的更新及加入新的功能

    几乎支持所有主流与非主流数据库

    高速的执行效率，

    丰富的函数库

    PHP从PHP4开始就引入了面向对象的特性，到PHP6 已经很完善了。
