    • Cgicc的安装

    站长原创，版权所有ITEEDU，2011-07-24

    在ubuntu下边，先在官方网站http://www.gnu.org/software/cgicc/上下载了源码包，解压缩到任何一个目录下边，然后通过一下步骤进行安装：

    （1） ./configure 

    （2） sudo make

    （3） sudo make install

    注意：

    （1）./configure可以通过参数—prefix来指定安装的目录，在此我们选择了默认的目录，那么安装完毕后，库默认安装在/usr/lib中，头文件默认安装在/usr/include中。

    （2）在第二步的时候会出一些小错误：

    大概的错误提示是：找到不到doc文件之类的，我以为就安装不了了呢，然后就放弃了后边的安装了，耗了两天，后来还是硬着头皮，编译出错，可能不是致命的，所以就进行了第三步，结果安装没有出错。

    因为我们下边进行cgicc开发的时候，是在eclipse中开发的，所以也就没有设置环境变量，只需要下边在eclipse中进行添加即可，呵呵。

    还有，我下载的是cgicc-3.2.9版本的，因为之前在编译时，出现过小问题，所以也试过其他的版本，但是问题更大，于是最后还是采用了cgicc-3.2.9版本，我估计问题就是编译器版本的影响。

    其他先不管了，童鞋们，如果你在编译时，出现了找不到：文件doc类似的错误，不妨走下去，这种错误不会影响最终的安装和使用的。呵呵
