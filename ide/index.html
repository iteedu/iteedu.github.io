    Visual Studio （WINDOWS）

     

    编程工具IDE

    站长原创，版权所有ITEEDU

    当前常见编程工具IDE：

    Eclispe（通用）

    由IBM开发的开源编程工具，由于很好的插件技术，很容易扩展，几乎能支持任何开发的需要。

    支持JAVA，C/C++，PYTHON，WEB......

    Netbeans（通用）

    类似Eclipse，Netbeans是由JAVA社区开发的开源IDE。也是有很好的插件技术，很容易扩展。有人说Eclispe像LINUX，Netbeans像MAC。

    支持JAVA，C/C++，PYTHON，WEB......

    Visual Studio （WINDOWS）

    微软.NET平台的开发环境，很强大的功能集合，支持所有微软推出的技术的开发。

    支持VC++.NET，C#，VB.NET，J#，WEB（ASP.NET）......
