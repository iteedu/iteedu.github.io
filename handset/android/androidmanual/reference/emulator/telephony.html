    • 电话仿真

    Android模拟器包括自己的GSM效仿调制解调器，可让您模拟电话 职能的模拟器。例如，您可以输入模拟电话和建立/终止数据 连接。该Android系统处理模拟电话完全一样，将实际的要求。 模拟器不支持音频电话本新闻稿中。

    您可以使用控制台访问模拟器的电话功能。 在连接到控制台，您可以使用

    gsm <call|accept|busy|cancel|data|hold|list|voice|status>

    to invoke telephony functions.

    gsm命令subcommands支持下表中列出.

    子命令 描述 注释
    call <phonenumber> 模拟输入电话 <phonenumber>.  
    accept <phonenumber> 接受入站的电话<phonenumber>和变化电话的状态为"active". 你可以切换状态到 "active" 只有当前状态是 "waiting" 或 "held".
    busy <phonenumber> 拒绝出站的电话<phonenumber>和变化电话的状态为"busy". 你可以切换状态到 "busy" 只有当前状态是s "waiting".
    cancel <phonenumber> 终止一个入站或出站的电话号<phonenumber>.  
    data <state> 改变现在的GPRS数据连接 <state>. Supported <state> values are:
  • unregistered -- 没有网络连接的情况
  • home -- 在本地网，非漫游
  • roaming -- 在漫游网络
  • searching -- 搜索网络
  • denied -- 仅紧急电话
  • off -- 同为'未经注册'
  • on -- 一样作为 'home'
    		•
    hold 改变当前状态到 "held". 你可以切换状态到 "held" 只有当前状态是"active" or "waiting".
    list 列出所有呼入和呼出电话和他们的国家.  
    voice <state> 改变现在的GPRS数据连<state>. Supported <state> values are:
  • unregistered -- 没有网络连接的情况
  • home -- 在本地网，非漫游
  • roaming -- 在漫游网络
  • searching -- 搜索网络
  • denied -- 仅紧急电话
  • off -- 同为'未经注册'
  • on -- 一样作为 'home'
    		•
    status Report the current GSM voice/data state. Values are those described for the voice and data commands.
