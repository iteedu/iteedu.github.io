    Android 移动手机模拟器

    Android SDK 内置了一个移动手机模拟器 -- 一个运行在你的计算机上的虚拟移动设备. 这个模拟器能使你设计，开发，测试你的Android应用程序，而不需要一个物理设备.

    Android 移动手机模拟器模拟了典型的移动设备所有的典型硬件和软件特征，除了不能接听和拨打真正的电话. 它提供了多种导航和控制键，当你使用鼠标或键盘"press"的时候，就会为你的应用程序产生事件. 它也提供一个屏幕来显示你的应用程序和其他正在运行的Android应用程序.

    为了帮助你模拟和测试你的应用程序，模拟器让你的应用程序可以使用Android平台的各种服务去调用其他的应用程序， 访问互联网，播放音频和视频，存储和读取数据，通知用户，图片转换和主题.

    模拟器同时也提供丰富的调试功能，例如一个可以把记录内核输出日志的控制台，仿真程序中断( 例如一个短信息或电话来了)，和仿真延迟效果以及中途退出数据通道.
