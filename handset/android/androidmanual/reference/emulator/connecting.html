    为了让一个模拟器实例与另一个实例通信，则必须成立 必要的网络重定向，如下所示.

    假设您的环境是

    • A 是你开发机器
    • B is 是你第一个模拟器实例，运行在 A
    • C is 是你第二个模拟器实例，也运行在 A

    你想运行一台服务器的B ，C和B连接，这里是你如何设置：

    1. 设置服务器B ，监听 10.0.2.15:<serverPort>
    2. 在B的控制台，成立了一个由重定向 的A:localhost:<localPort>到 B:10.0.2.15:<serverPort>
    3. 对C ，有客户端连接到10.0.2.2 ：<localPort>

    例如，如果您想运行一个HTTP服务器，您可以选择 的<serverPort> 80和 <localPort> 8080:

    • B 监听 10.0.2.15:80
    • 在 B 的控制台, 输入 redir add tcp:8080:80
    • C 连接 10.0.2.2:8080
