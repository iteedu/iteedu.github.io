    • 9-patch绘图

    9-patch绘图工具能让你轻松的用一个所见即所得编辑器来创建一个 NinePatch图形。

    想了解什么是NinePatch图形以及他们的工作原理，请阅读Available Resource Types 文档中关于Nine-patch的章节


    下面是一个使用Draw 9-patch工具来创建一个NinePatch图形的快速指南。你需要先准备一幅想要用来创建NinePatch图形的PNG格式图片。

    1. 在计算机上，从draw9patchSDK /tools目录下启动应用。
    2. 将你的PNG图片拖入Draw 9-patch窗口 (或者通过点击 File > Open 9-patch...来选择图片). 你的工作区就可以打开了。

      左边的面板是你的绘画区域，你可以在其中编辑伸缩补丁和内容区域中的线条。 右边的面板为预览区域，当你的图片伸缩时可以在这进行预览。

    3. 按着1像素一周能在内容区域中画出一条已定义的线条。右键单击Right-click (在mac电脑上按住Shift点击)来擦除画的线条。
    4. 完成后，选择File > Save 9-patch...

      你的图片将会保存在名为.9.png的文件中

    Note: 当你在单像素边界上添加了一个普通的PNG(*.png)文件后，你可以在其中来画出内容。 没有绘图区域增加时，以前保存的9-patch 文件 (*.9.png) 同样会被加载，因为他们已经存在了

    控制选项包括：

    • Zoom: 调整绘图区域中放大视图的放大倍率。
    • Patch scale: 调整预览区域图片的尺度。
    • Show lock: 鼠标经过不可绘图区域图片时显示。
    • Show patches: 在绘图区域预览伸缩图(粉色的表示为一个伸缩块)。
    • Show content: 在内容区域中高亮显示预览图。(区域中的紫色表示可选内容)。
    • Show bad patches: 在修补区域添加红色边界，拉伸图片时可能因此产生一种假象。如果你除去 所有的坏补丁块，就会显得图像黏连在了一起。
