    一、结构

    public abstract class ResourceCursorAdapter extends CursorAdapter
         
java.lang.Object
         android.widget.BaseAdapter
                   android.widget.CursorAdapter
                            android.widget.ResourceCursorAdapter

    直接子类

     SimpleCursorAdapter

    二、概述

    这是一个简单的适配器，通过指定一个定义了视图UI的XML文件来创建视图。

    三、构造函数

     public ResourceCursorAdapter (Context context, int layout, Cursor c)

    构造函数

    参数

    1. Context 与ListView相关的正在运行的 SimpleListItemFactory上下文
    2. layout 一个定义了列表项视图的布局文件资源ID，这个布局同时定义列表项视图和下拉视图，直到你重写它们。
    3. c 获取数据的游标
    public ResourceCursorAdapter (Context context,int layout, Cursor c, boolean autoRequery)

    构造函数

    参数

    1. Context 与ListView相关的正在运行的 SimpleListItemFactory上下文
    2. layout 一个定义了列表项视图的布局文件资源ID，这个布局同时定义列表项视图和下拉视图，直到你重写它们。
    3. c 获取数据的游标
    4. autoRequery 如果此参数为true，当适配器的数据发生变化的时，适配器会调用游标的requery()方法，使最新的数据始终显示。

    四、公共方法

     public View newDropDownView (Context context, Cursor cursor, ViewGroup parent)

    生成一个新的下拉视图来控制游标指向的数据

    参数

    1. context 应用程序全局信息接口（应用上下文）
    2. cursor 获取数据的游标，它已经移动到正确的位置
    3. parent 与新视图相关联的上级视图

    返回值

    新创建的视图

    2. cursor 获取数据的游标，它已经移动到正确的位置
    3. parent 与新视图相关联的上级视图
